
# 🎵 Music API

A simple REST API for managing a music collection, built with pure Node.js core modules (no external frameworks).

## Features

- ✅ Create new songs (POST)
- ✅ Read all songs (GET)
- ✅ Read single song by ID (GET)
- ✅ Update existing songs (PUT)
- ✅ Delete songs (DELETE)
- ✅ Persistent storage using JSON file
- ✅ Only Node.js core modules (http, fs, path)

## Technologies Used

- Node.js (http module for server creation)
- Node.js (fs module for file handling)
- JSON file storage
